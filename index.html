How to play: Use your arrow keys to move the tiles. When two tiles with the same number touch, they merge into one!

2048 3x3 is an extended version of the classic 2048 game, featuring a 3x3 grid instead of the standard 4x4 board. This variation allows for a more strategic approach to merging tiles, making it a great option for players who enjoy longer gameplay sessions.

This 3 x 3 fast paced version by Kaloo Dinaz was cloned from the "non-official" 2048 game authored by Kaloo Dinaz.
